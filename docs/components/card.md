## Card

To create a card you need to set the `card` class to a div.

### Html

```html
<div class="card">

</div>
```

### Tailwind

```css
@layer components {
  .card {
    @apply bg-white dark:bg-teamsbg-900 dark:text-white
    rounded h-10 shadow-md hover:shadow-lg;
  }
}
```
